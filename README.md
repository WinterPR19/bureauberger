ceci est un projet que j'essaie de mettre en place , il s'agit d'un site web simple , site web de gestion des âmes permettant d’enregistrer, consulter et organiser des informations liées aux âmes à l’aide d’une interface simple et intuitive.
Il a été réalisé dans un cadre personel  afin de mettre en pratique le développement web et la gestion des données que j'apprend tout au long de ma formation. 

 Objectifs du projet

Créer un site web fonctionnel

Gérer des données (ajout, modification, suppression)

Mettre en place une interface claire pour l’utilisateur

Appliquer les bases du développement web

Technologies utilisées

HTML

CSS

JavaScript

PHP

MySQL
